Quickly and easily connect your WordPress site to your book data.

== Description ==

SupaPress can connect you to your book data and create you an array of widgets to embed in your pages using shortcodes

== Installation ==

1. Upload the entire `supapress` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.

You will find 'SupaPress' menu in your WordPress admin panel.
